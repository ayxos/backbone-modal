About

Create a custom backbone-wizard.

  • must be used as a requireJS component.
  • need bootstrap as dep.

Really simple, just name it and set a save action

...

var Backbone_modal = require('path/to/backbone/modal/component');
var AnotherView = require('path/to/another/view')

...

new Backbone_modal(AnotherView);

...

RequireJs Config file

...

shim: {
    'path/to/backbone-simple-modal/lib':{
      deps:['underscore', 'bootstrap' , 'backbone']
    }
}

...

Get backbone-modal.js

Get the backbone-simple-modal via bower:

bower install backbone-simple-modal
